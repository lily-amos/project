Background

The woody plant vegetation structure dataset (National Ecological Observatory Network (NEON) 2019) contains the quality-controlled, native sampling resolution data from in-situ measurements of live and standing dead woody individuals and shrub groups, from all terrestrial NEON sites with qualifying woody vegetation. This protocol adopts guidelines established by the “Forest Inventory and Analysis National Core Field Guide. Volume i: Field Data Collection Procedures for Phase 2 Plots” (2012) for measuring tree species.

Our analyses focuses on the relationship between individual stem heights and diameter and how that relationship varies across growth forms.

Data Preparation

The analytical dataset was prepared in the following ways:

  • Combined all data files into a single table
  • Geolocated individual stems by combining individual dataset with plot level data

Full data preparation steps can be found in the data-raw/individuals.R script.

Summary Statistics

Prepared data were also subset to columns of interest stem_diameter, height and growth_form and records where growth_form values were not "liana" or missing.

Code 
## Setup ----
## Load libraries
library(ggplot2)
library(dplyr)

## Load data
individual <- readr::read_csv(
  here::here("data", "individual.csv")
  ) %>% 
  select(stem_diameter, height, growth_form)

# Subset analysis data ----
analysis_df <- individual %>%
  filter(complete.cases(.), 
         growth_form != "liana")

## Order growth form levels
gf_levels <- table(analysis_df$growth_form) %>%
  sort() %>%
  names()

analysis_df <- analysis_df %>%
  mutate(growth_form = factor(growth_form,
                              levels = gf_levels
  ))

Analysis

Relationship between log height and stem diameter

Initially we fit a linear model of form log(stem_diameter) as a function of log(height).

lm_overall <- lm(
  log(stem_diameter) ~ log(height),
  analysis_df
)
`geom_smooth()` using formula = 'y ~ x'

See Figure 3, Table 2 and Table 3 for results.

Growth form level analysis

We also fit a model with an interaction term with growth form included.

lm_growth <- lm(
  log(stem_diameter) ~ log(height) * growth_form,
  analysis_df
)
`geom_smooth()` using formula = 'y ~ x'

See Figure 4, Table 4 and Table 5 for results.

Summary

Our results agree with findings in the literature (Haase 2008; THORNLEY 1999; Cannell 1984).
